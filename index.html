RELEMBRAR

Uma ontologia para pessoas desaparecidas.

VER MAIS

A ontologia Relembrar foi criada para modelar o domínio de pessoas desaparecidas. Sua estrutura permite descrever características físicas, dados pessoais e dados referentes ao desaparecimento de uma pessoas. A ontologia Relembrar é uma das primeiras iniciativas de criação de um modelo ontológico para representar o contexto de pessoas desaparecidas.
